## 26. Remove Duplicates from Sorted Array (Medium)
[tag] two pointers
- two pionters both start from the left. 
```python
'''
Two pointers:
0 1 0 1 2 2 4
  l
      r  
'''

class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        if not nums:
            return
        l, r = 0, 0
        while r < len(nums):
            if nums[l] != nums[r]:
                l += 1
                nums[l], nums[r] = nums[r], nums[l]
            r += 1
        return l + 1
```
