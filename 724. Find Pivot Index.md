## 724. Find Pivot Index

- use one pointer to move and adjust left sum and right sum in each iter. 

```python
class Solution:
    def pivotIndex(self, nums: List[int]) -> int:

        for i in range(len(nums)):
            if i == 0:
                l_sum = 0
                r_sum = sum(nums[1:])
            else:
                l_sum += nums[i-1]
                r_sum -= nums[i]
            
            if l_sum == r_sum:
                return i;
        return -1
```
